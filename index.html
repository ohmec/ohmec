Open History Map Exploration

The OHMEC project has a goal to create an open geographical database of historical representation of political boundaries. This demonstration shows a simple rendering of data in the North American arena. A full featured rendering would require tiling from the database indexed with time and geography.

The OHMEC project is being developed in a Github repository. Participation is welcome, see that repo for more information on the state of the project.

A rendering of the current state of the project is reflected at our Website.

Arguments can be passed to the URL to modify timeline parameters and default viewing perspective. These arguments are shown below, and are set as for example http://ohmec.net?parameter=value&parameter=value.

parameter argument description
startdatestr YYYY:MM:DD Sets timeline slider start date to MM/DD/YYYY
enddatestr YYYY:MM:DD Sets timeline slider end date to MM/DD/YYYY
curdatestr YYYY:MM:DD Sets timeline slider current date to MM/DD/YYYY
lat N Sets center latitude to value N (range -90 to 90)
lon N Sets center longitude to value N (range -180 to 180)
z N Sets zoom value to N (range 1 to 18)
