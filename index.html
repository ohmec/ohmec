Open History Map Exploration

The OHMEC project has a goal to create an open source geographical database of a historical representation of political boundaries throughout all recorded history. It is very much a work in progress. This early demonstration shows a rendering of data in the North American arena using Leaflet to display geographical polygons from the database, overlaid on top of a modern map.

The OHMEC project is being developed openly in a Github repository. Participation is welcome, see that repo for more information on the management of the project. Also see this slidedeck for more insight into the project. A specification of the extended GeoJSON format that underpins the historical database is provided here.

Holding the shift key down and creating a bounding box will redefine the current zoom window. Clicking on a polygon will hold the Feature Information for further inspection.

Visualization and control can be modified by key press events. The following keys pressed after click-selecting the map pane have a particular effect.

key effect
0 Modify underlying base map to use unlabeled relief map (default)
1 Modify underlying base map to use unlabeled darker world imagery map
2 Modify underlying base map to use unlabeled light physical imagery map
3 Modify underlying base map to use unlabeled white relief map
4 Modify underlying base map to use unlabeled Stamen terrain map
5 Modify underlying base map to use labeled street map
6 Modify underlying base map to use unlabeled watercolor "paint" map
s Toggles the "smart step" button feature. When enabled (default true), the step + and step - buttons move the timeline slider forward or backward to the next time where a change in the database is visible in the current viewpoint. When false, the step buttons simply move to the next or previous change in the database. NOTE the user must click on the map once to indicate focus on it (vs other browser items) before any keyboard event.

Arguments can be passed to the URL to modify timeline parameters and default viewing perspective. These arguments are shown below, and are set as for example http://ohmec.net?param1=val1&param2=val2.

parameter argument description
startdatestr YYYY:MM:DD Sets timeline slider start date to MM/DD/YYYY
enddatestr YYYY:MM:DD Sets timeline slider end date to MM/DD/YYYY
curdatestr YYYY:MM:DD Sets timeline slider current date to MM/DD/YYYY
lat N Sets center latitude to value N (range -90 to 90)
lon N Sets center longitude to value N (range -180 to 180)
z N Sets zoom value to N (range 1 to 18)
smartstep on|off Enables or disables the smart step feature.
